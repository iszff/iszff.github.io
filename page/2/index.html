我的思路

栈和队列是相反的，先后入两个不同的栈相当于入一个队列，

比如Q{1，2，3，4}，head =1，tail=4，

S1=[4，3，2，1]，top=4，bottom=1，让S1中的元素一个个弹出至S2中，可以得到S2=[1，2，3，4]，top=1，bottom=4。

对于刚建立的空CQueue，返回值为null，

需要实现appendTail和deleteHead，

appendTail返回值是null，需要把新tail元素压入S1中位于S1top，把S2全部更新才可以实现把tail放至S2的bottom，但是这样的话每次appendTail都要产生一个全新的S2，会开一块新的内存空间，并且出栈入栈，那复杂度。。。。不不不，不用啦哈哈，因为有两个栈一个出清空，另一个入填满，来回倒来倒去就可以啦!

所以过程是S1=[4，3，2，1]，S2=[]

S1=[3，2，1]，S2=[4]；S1=[2，1]，S2=[3，4]…S1=[]，S2=[1，2，3，4]

appendTail

初始S1=[]，S2=[1，2，3，4]；S2出栈至S1，元素顺序全部倒置，变为S1=[4，3，2，1]，S2=[]，此时把tail=5入栈到S1，变为S1=[5，4，3，2，1]，S2=[]；再次让S1中的元素一个个弹出至S2中，变为S1=[]，S2=[1，2，3，4，5]，实现了appendTail功能。

输出null。

每次append后重新倒了回去，多花了一倍时间20211028.

deleteHead

直接pop掉S2的顶部元素即可。输出栈顶元素。

用size作为循环的终止条件，size是会随着每一次的pop、push操作更新的。

20211029剑指 Offer 35. 复杂链表的复制 - 力扣（LeetCode） (leetcode-cn.com)

看到题目的感觉是直接记录下一个节点的指针和random的值，但是random的类型是node型指针。怎么把*和出现的顺序的index结合起来呢？

官方题解

对一个特殊的链表进行深拷贝。如果是普通链表，可以直接按照遍历的顺序创建链表节点。

本题中因为随机指针的存在，当我们拷贝节点时，「当前节点的随机指针指向的节点」可能还没创建，因此我们需要变换思路。

一个可行方案是，我们利用回溯的方式，让每个节点的拷贝操作相互独立。

对于当前节点，我们首先要进行拷贝，然后我们进行「当前节点的后继节点」和「当前节点的随机指针指向的节点」拷贝，拷贝完成后将创建的新节点的指针返回，即可完成当前节点的两指针的赋值。

读完这个话也想不到应该怎么做呢？

具体地，我们用哈希表记录每一个节点对应新节点的创建情况，

遍历该链表的过程中，我们检查「当前节点的后继节点」和「当前节点的随机指针指向的节点」的创建情况。

如果这两个节点中的任何一个节点的新节点没有被创建，我们都立刻递归地进行创建。

当我们拷贝完成，回溯到当前层时，我们即可完成当前节点的指针赋值。注意一个节点可能被多个其他节点指向，因此我们可能递归地多次尝试拷贝某个节点，为了防止重复拷贝，我们需要首先检查当前节点是否被拷贝过，如果已经拷贝过，我们可以直接从哈希表中取出拷贝后的节点的指针并返回即可。

重叠子问题。

最优子结构。

正确的状态转移方程。

明确 base case -> 明确「状态」-> 明确「选择」 -> 定义 dp 数组/函数的含义。

但凡遇到需要递归的问题，最好都画出递归树，这对你分析算法的复杂度，寻找算法低效的原因都有巨大帮助。

A Deeply-initialized Coarse-to-fine Ensemble of Regression Trees for Face Alignment用于人脸配准的深度初始化的粗到细的回归树集合

Abstract

在本文中，我们提出了DCFE，一种基于粗到细的 回归树集合(ERT)的实时面部地标回归方法。

我们使用简单的卷积神经网络(CNN)来生成地标位置的概率图。这些数据通过ERT回归器进一步细化，其中ERT回归器是通过将3D人脸模型拟合到地标地图上进行初始化的。

ERT由粗到细的结构使我们能够解决零件变形的组合爆炸问题。利用3D模型，我们还解决了其他关键问题，如鲁棒回归器初始化、自遮挡以及同时进行正面和侧面人脸分析。

在实验中，DCFE在AFLW、COFW和300W私有和公共数据集中取得了最好的报告结果。

Introduction

面部关键点检测是人脸图像分析的一个初步步骤，如验证与识别[25]、属性估计[2]等。大型注释数据集的可用性最近鼓励了这一领域的研究，并带来了重要的性能改进。然而，这仍然是一个具有挑战性的任务，特别是当面孔遭受较大的姿势变化和部分遮挡。

在最近的300W基准测试中，表现最好的都是基于深度回归模型20,23,30,33。由于深度网的感受野很大，这些方法最突出的特点是鲁棒性。然而，在这些模型中，很难加强面部形状的一致性或估计自遮挡。

另一方面，基于ert的模型[6,7,18,24]很容易并行化，并且在它们的估计中隐含地强加了形状一致性。它们比深度模型更有效，正如我们在实验中所演示的(见图4)，在良好的初始化情况下，它们也非常准确。

在本文中，我们提出了一种混合方法，称为深度初始化粗-细集成(DCFE)。它使用一个简单的CNN来生成地标位置的概率图，在没有全局强加形状的情况下获取关于单个地标的位置信息。然后我们拟合一个三维人脸模型，从而增强全局人脸形状先验。这是粗-细ERT回归分析的起点。拟合的3D人脸模型为回归器提供了有效的初始形状和有关地标能见度的信息。从粗到细的方法让ERT很容易地解决非刚性零件所有可能变形的组合爆炸，同时施加部分形状优先。该方法可实时运行(32 FPS)，并在AFLW、COFW和300W私有和公共数据集中提供最佳的报告结果。

Related Work

人脸对齐是一个二十多年来备受关注的研究课题。初步成功的结果是基于2D和3D生成方法，如活动外观模型(AAM)[8]或3D变形模型[4]。最近的鉴别方法基于两个关键思想:根据当前形状估计[12]索引图像描述，或者是使用一个回归器，该回归器的预测位于由训练面部形状[7]所构成的子空间上，这就是所谓的级联形状回归器(CSR)框架。Kazemiet al.[18]通过提出一个实时的回归树集成来改进原来的级联框架。Renet al.[24]使用本地二进制特性将性能提高到3000 FPS。Burgos-Artizzuet al.[6]包括遮挡信息。Xionget等[31,32]使用SIFT特征并学习一个线性回归器，将搜索空间划分为具有相似梯度方向的单个区域。总体而言，CSR方法对回归过程的起点非常敏感。最近工作的一个重要部分是围绕如何找到好的初始化[38,37]。在本文中，我们使用由CNN生成的标定的概率图来寻找CSR的鲁棒起点。

目前最先进的人脸对齐方法是基于cnn的。Sun等人[26]是应用三层CNN来获得准确地标估计的先驱。zhang等人[36]提出了一种处理人脸对齐和属性分类的多任务解决方案。Lvet等人的[23]使用全局和局部人脸部分回归器进行细粒度面部变形估计。Yuet al.[34]变换地标而不是用于细化级联的输入图像。Trigeorgiset al.[27]和Xiaoet al.[30]是第一个将CSR的特征提取和回归步骤融合到端到端训练的递归神经网络的方法。Kowalskiet al.[20]和Yanget al.[33]在Menpo比赛[35]中名列前茅。两者都使用全局相似变换对地标位置进行归一化，然后分别使用基于vgg和堆叠沙漏网络回归最终形状。深度神经网络的大感受野使得这些方法对人脸旋转、缩放和变形具有高度的鲁棒性。然而，目前还不清楚如何将面部形状的一致性强加到估计的地标集合上。此外，为了达到精度，他们求助于深度模型的级联，逐步细化估计，从而增加计算需求。

基于3D人脸模型的作品也越来越多。在最简单的情况下，它们将一个均值模型拟合到估计的图像地标位置[19]或联合回归人脸的姿态和形状[17,29]。这些方法提供3D姿势信息，可用于估计专门针对给定的头部方向地标自遮挡或训练简单的回归器。然而，构建和拟合三维人脸模型是一项艰巨的任务，在目前的基准测试中，完整的3D方法的结果不如上述描述的那样好。

我们的建议试图利用前面方法的最佳特性。使用基于cnn的初始化，我们继承了深度模型的鲁棒性。像简单的3D方法一样，我们拟合一个刚性3D人脸模型来初始化ERT，并估计全局人脸方向以解决自遮挡。最后，我们在粗到细的框架内使用ERT来实现精度和效率。

Deeply Initialized Coarse-to-Fine Ensemble

在本节中，我们介绍了深度初始化的粗到细集成方法(DCFE)。该算法主要包括两个步骤:基于cnn的刚性脸位姿计算和基于ert的非刚性脸变形估计，如图1所示。

3.1 Rigid pose computation

基于ert的回归器需要一个可接受的初始化来收敛到一个好的解决方案。

我们提出使用人脸地标位置概率图生成貌似合理的形状初始化候选。我们修改了Honariet al.的[16]RCN，引入了一个loss函数来处理丢失的地标，从而实现了半监督训练。我们训练这个CNN获得一组概率映射，P(I)，表示输入图像中每个标定点的位置(见图1)。每个平滑概率映射的最大值决定了我们的初始路标位置。请注意，在图1中，这些预测对遮挡很敏感，可能不是一个有效的面部形状。与典型的基于cnn的方法相比，例如[33]，我们的CNN更简单，因为我们只需要粗略估计地标位置。

为了以一个可信的脸开始ERT，我们通过拟合一个刚性3D头部模型到估计的2D地标位置来计算初始形状。为此，我们使用了david al.[10]提出的softPOSIT算法。因此，我们使用估计的刚性变换将三维模型投影到图像上。这为ERT提供了目标人脸的比例、平移和3D姿态的粗略估计(见图1)。

让x^0=g_0(P(I))为初始形状，初始化函数g_0处理输入image I后的输出。在这种情况下，x^0是一个L×2向量与L 2d地标坐标。通过初始化，我们确保x0是一个有效的脸形状。这保证了算法下一步的预测也将是有效的脸型[7]。

3.2 ERT-based non-rigid shape estimation

  • $P(I)$ a set of probability maps经过CNN得到，之后经过高斯平滑滤波器GS，再经过Max函数，得到粗糙的标定，由于对于遮挡的敏感性不一定是一个有效的脸型。
  • $x^0=g_0(P(I))$是过POSIT得到的,$x^0$是一个$L\times2$的向量，$x^0$是一个有效的脸型，它作为ERT的初始化。
  • $S={s_i}^N_{i-1}$ the set of train face shapes，$s_i={I_i,x^g_i,v^g_i,w^g_i,x^g_i}$，对于每一个训练形状$s_i$有$I_i$训练图片，真值形状$x^g_i$,真值可见标定$v^g_i$,带标注的地表标签$w^g_i$(1代表标注了的，0代表丢失的)。
  • 初始化形状为$x_i^0$,gt即目标形状为$x_i^g$,是一个$L\times2$向量。$L\times1$向量$v_i^g$是每个标定点可见性的二进制标签，如果对于第$k$个组件，$v_g(k)=1$，表示对于第$k$个地标是可见的。
  • shape-indexed features $\phi(P(I_i),x^t_i,w_i^g)$依赖于在图片$I_i$关键点的当前的形状和它们是否被标注了，即$x_i^t$和$w_i^t$。
  • 把回归过程分为$T$个阶段，对于第$t$阶段学习$K$个回归树集合，$C_t(f_i)=x^{t-1}+\sum^K_{k=1}g_k(f_i)$，$f_i=\phi(P(I),x^{t-1}_i,w_i^g)$,$x^j$是第$j$个阶段估计出的标定点的坐标，在第一个阶段中为$x^0$，为了训练整个ERT，使用$S$中的N个训练样本生成一个增广训练集，$S_A$的基数是$N_A$=$|S_A|$。
  • 对于每个训练形状$s_i$，通过改变初始形状生成了附加的训练样本，从平滑的概率图中随机抽取新的候选地标位置，以生成新的初始形状。
  • 每个初始化形状可以逐步被估计的形状和可见性增量$C^v_t(\phi(P(I_i),x^{t-1}i,w^g_i)$，$x_i^{t-1}$代表了第$i$个样本的当前形状，$C^v_t$训练为了最小化关键的错误，但在每个叶节点上，要输出平均形状和训练形状的可见性，定义$U{t-1}={(x_i^{t-1},v_i^{t-1})}^{N_A}_{i=1}$是所有训练数据的所有当前形状和相应可见性向量的集合。

与传统的ERT方法相比，我们的集成更简单。这将需要更少的树，因为我们只需要估计非刚性面的形状变形，因为在前面的步骤中已经估计了3D刚性组件。下面，我们将详细介绍我们的ERT。

Initial shapes for regression

ERT中起始点的选择是达成一个好的解决方案的基础。最简单的选择是地面真值训练形状的平均脸。然而，这样一个糟糕的初始化导致错误的对齐结果在测试图像与大的姿态变化。可供选择的策略是使用不同的初始化多次运行ERT，并使用[6]的中值，使用其他地面真实形状进行初始化，或随机变形初始形状。

在我们的方法中，我们使用3.1节描述的算法初始化ERT，该算法为初始化提供了一个鲁棒的近似形状(见图2)。因此，ERT只需要估计人脸姿态的非刚性分量。

Feature Extraction

ERT效率取决于特征提取的步骤。一般来说，描述符特征，如SIFT[31,38]，改善了人脸对齐结果，但与简单的特征，如普通像素值差[7,6,18,24]相比，计算成本更高。在我们的例子中，一个简单的特征就足够了，因为形状标志接近它们的地面真实位置。

在DCFE中，我们使用概率映射P(I)来提取级联的特征。为此，我们选择一个与其相关的概率图Pl(I)。**该特征计算为Pl(I)中来自一个FREAK描述符模式[1]**的两个像素值之间的差值。我们的功能类似于Lee et al.的[21]。然而，我们的定义是基于概率映射P(I)，而不是图像I。我们让训练算法在每次迭代中选择信息量最大的地标和像素对。

Learn a coarse-to-fine regressor

为了训练t级回归器，$C_t^v$，我们拟合ERT。因此，我们的目标是顺序地学习一系列弱学习者，以贪婪地最小化回归损失函数:

$L_t(S_A,F_A,U_{t-1})=\sum_{i=1}^{N_A}||w_i^g\odot (x_i^g-x_i^{(t-1)}-\sum_{k=1}^{K}g_k(f_i))||^2$

$\odot$是Hadamard product。

XGbOST

在训练全局人脸地标回归器时，一个关键问题是缺乏显示所有可能的人脸部分变形组合的示例。因此，这些回归器很快地过拟合，并对训练集中不存在的部分变形组合泛化得很差。为了解决这个问题，我们引入了粗到细的ERT体系结构。目标是能够处理在训练中看不到的脸部变形的组合。

目标是能够处理在训练中看不到的脸部变形的组合。单个整体回归器无法估计这些局部变形(见图3b和图3c的差异)。我们的算法是不可知的部分的数量或级别的粗到细的估计。算法2详细地训练了P个脸部分回归器(每个都有一个子集的标定点)，以建立一个粗到细的回归器。注意，$x^0_i$和$v^0_i$是来自最后一个回归量输出(例如，前一部分回归量或前一个完整阶段回归量的形状和可见性向量)。在我们的实现中我们使用$P=1$(全部的标定)，在first K1回归器和last K2回归器中，零件的数量增加到$P=10$(左/右眉毛，左/右眼睛，鼻子，上/下嘴巴，左/右耳朵和下巴)，如图3c中所有由线连接的部件。

Fit a regression tree

考虑标注的地标标签，第k回归树的训练目标是最小化残差平方和

$\epsilon_k=\sum_{i=1}^{N_A}||r_i^k||^2=\sum_{i=1}^{N_A}||w_i^g\odot(x_i^g-x_i^{k-1})||^2$

我们通过递归地将训练集分解为左(l)和右(r)子节点来学习每棵回归二叉树。树节点分裂函数的设计目的是使所选地标中的残差平方和最小化。训练回归树节点我们随机生成一组候选分裂方程，每个包含四个参数$θ=(τ,p_1, p_2, l)$， $p_1$和$p_2$是在$x_i^{k-1}$中第$l$个关键点附近的固定FREAK结构。相对应的第$i$个样本的特征值$θ$，有$f_i(\theta)=P^l(I_i)[p1]-P^l(I_i)[p2]$为给定地标在map上的的概率值之差。最后，根据特征值计算分裂函数阈值$fi(θ)$> τ。

给定一个节点上的$N\sub S_A$训练样本集合，对第k个树，包括寻找参数$\theta$可以最小化$E_k(N,\theta)$

$\arg \min\limits_{\theta} E_k(N,\theta) =\arg \min\limits_{\theta} \sum\limits_{b\in{l,r }} \sum \limits_{s\in N_{\theta,b}}||r_s^k-\mu_{\theta,b}||^2$

其中$N_{\theta ,l}$和$N_{\theta,r}$分别为由$θ$归纳而发送到左右子节点的样本，平均残差$\mu_{\theta,b}$给出了一个候选分裂函数和一个训练数据子集

$\mu_{\theta,b}= \frac{1}{|N_{\theta,b}|}\sum \limits_{s \in N_{\theta,b}}r^k_s $

一旦我们知道了最优分割，每个叶节点存储了平均残差$\mu_{\theta,b}$作为任何到达那个叶子的例子的回归的输出。

DCFE在300W私有和公共数据集排第4，3.24。

  1. 评价：

    提出了DCFE Deeply-initialized Coarse-to-fine Ensemble of Regression Trees，一种基于粗到细的 回归树集合(ERT)Ensemble of Regression Trees 的实时面部地标回归方法。

  2. 针对的问题：

    1. 解决零件变形的组合爆炸问题 the combinatorial explosion of parts deformation。
    2. 解决鲁棒性地回归器初始化、自遮挡问题以及同时进行正面和侧面人脸分析。

  3. 本文的目的：

  4. 实现的方法：

    基于cnn的刚性脸位姿计算和基于ert的非刚性脸变形估计

    • $P(I)$ a set of probability maps经过CNN得到，之后经过高斯平滑滤波器GS，再经过Max函数，得到粗糙的标定，由于对于遮挡的敏感性不一定是一个有效的脸型。
    • $x^0=g_0(P(I))$是过POSIT得到的,$x^0$是一个$L\times2$的向量，$x^0$是一个有效的脸型，它作为ERT的初始化。
    • $S={s_i}^N_{i-1}$ the set of train face shapes，$s_i={I_i,x^g_i,v^g_i,w^g_i,x^g_i}$，对于每一个训练形状$s_i$有$I_i$训练图片，真值形状$x^g_i$,真值可见标定$v^g_i$,带标注的地表标签$w^g_i$(1代表标注了的，0代表丢失的)。
    • 初始化形状为$x_i^0$,gt即目标形状为$x_i^g$,是一个$L\times2$向量。$L\times1$向量$v_i^g$是每个标定点可见性的二进制标签，如果对于第$k$个组件，$v_g(k)=1$，表示对于第$k$个地标是可见的。
    • shape-indexed features $\phi(P(I_i),x^t_i,w_i^g)$依赖于在图片$I_i$关键点的当前的形状和它们是否被标注了，即$x_i^t$和$w_i^t$。
    • 把回归过程分为$T$个阶段，对于第$t$阶段学习$K$个回归树集合，$C_t(f_i)=x^{t-1}+\sum^K_{k=1}g_k(f_i)$，$f_i=\phi(P(I),x^{t-1}_i,w_i^g)$,$x^j$是第$j$个阶段估计出的标定点的坐标，在第一个阶段中为$x^0$，为了训练整个ERT，使用$S$中的N个训练样本生成一个增广训练集，$S_A$的基数是$N_A$=$|S_A|$。
    • 对于每个训练形状$s_i$，通过改变初始形状生成了附加的训练样本，从平滑的概率图中随机抽取新的候选地标位置，以生成新的初始形状。
    • 每个初始化形状可以逐步被估计的形状和可见性增量$C^v_t(\phi(P(I_i),x^{t-1}i,w^g_i)$，$x_i^{t-1}$代表了第$i$个样本的当前形状，$C^v_t$训练为了最小化关键的错误，但在每个叶节点上，要输出平均形状和训练形状的可见性，定义$U{t-1}={(x_i^{t-1},v_i^{t-1})}^{N_A}_{i=1}$是所有训练数据的所有当前形状和相应可见性向量的集合。

    1. 使用简单的卷积神经网络(CNN)来生成地标位置的概率图，深度神经网络的大感受野对人脸旋转、缩放和变形具有高度的鲁棒性。

    2. 拟合一个三维人脸模型，通过拟合一个刚性3D头部模型到估计的2D地标位置来计算初始形状，增强全局人脸形状先验，并估计全局人脸方向以解决自遮挡。

    3. 通过ERT回归器进一步细化特征，从粗到细的方法可以处理局部形变，让ERT很容易地解决非刚性零件所有可能变形的组合爆炸。

    ERT方法的优点

    容易并行化,它们的估计中隐含地强加了形状一致性。在粗到细的框架内使用ERT来实现精度和效率。

EDA课程设计要求

我们的EDA课程计使用的是学院实验室的Altera试验箱，核心板是Cyclone II EP2C5Q208C8芯片。

课程设计的要求如下：

应用Verilog HDL硬件描述语言，设计一个基于FPGA的空气净化器程序，模拟空气净化器的运行，其功能为：

1 实现5档出风的风力控制，使用控制电机模拟
2 实现当前空气质量的检测，通过设置拨码开关模拟
3 实现当前状态的指示与显示
4 实现睡眠设置功能，在指定的设置时间后风量输出调节至1档
5 实现连续运行时间的记录，并进行更换空气过滤网提示报警
6 具有自动模式和手动模式选择，自动模式根据检测到的空气质量自动调节风速，手动模式通过按键设置风速档位
7 实现时钟功能，并且根据设定的时间自动关机
